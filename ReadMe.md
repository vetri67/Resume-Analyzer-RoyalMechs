# 🔥 Resume Analyzer for Royal Mechs

## 📌 Description
A GUI-based AI Resume Analyzer that evaluates resumes using rule-based AI logic.  
It provides score, feedback, and suggestions for improvement.

---

## 🚀 Features
- 🎨 Colorful modern UI (CustomTkinter)
- 🛡️ Custom RoyalMechs logo
- 📄 PDF resume upload
- 📊 Animated progress bar scoring
- 🤖 AI suggestions panel
- 😀 Emoji-based feedback system
- 📌 Section-wise analysis (Skills, Projects)

---

## 🧠 AI Concept
This application uses rule-based classification logic to simulate AI-based resume evaluation.

---

## 🛠️ Technologies Used
- Python
- CustomTkinter
- PyPDF2
- Pillow (Image processing)
- PyInstaller

---
